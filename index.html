개인정보처리방침

1. 수집하는 정보

본 서비스는 사용자 식별 정보를 수집하지 않습니다. 서비스 이용 기록(페이지 방문, 검색어 등)은 브라우저 로컬스토리지에만 저장됩니다.

2. 제3자 서비스

  • YouTube Data API v3 – Google LLC 서비스 약관이 적용됩니다.
  • Google AdSense – 광고 제공을 위해 쿠키를 사용할 수 있습니다.

3. 쿠키 및 로컬스토리지

광고 최적화를 위해 Google이 쿠키를 사용할 수 있습니다. 크레딧 및 캐시 데이터는 로컬스토리지에 보관됩니다.

이용약관

1. 서비스 목적

본 도구는 YouTube 공개 데이터를 기반으로 한 참고용 분석 서비스를 제공합니다.

2. 책임 제한

분석 결과는 참고용이며, 실제 채널 성과를 보장하지 않습니다. YouTube API 정책 변경에 따라 서비스가 변경될 수 있습니다.

3. YouTube API 약관

본 서비스는 YouTube API를 사용합니다. 이용 시 YouTube API 서비스 약관 및 Google 개인정보처리방침에 동의하는 것으로 간주됩니다.
