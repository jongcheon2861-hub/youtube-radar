개인정보처리방침

본 서비스는 사용자의 개인정보를 수집하지 않습니다. YouTube Data API를 통해 공개된 채널 정보만을 조회합니다.

광고 서비스(Google AdSense)는 사용자 경험 향상 및 서비스 운영을 위해 사용됩니다. Google의 개인정보처리방침이 적용됩니다.

  • 수집 정보: 없음 (비로그인 서비스)
  • 쿠키: 광고 표시 목적의 Google AdSense 쿠키만 사용
  • 로컬스토리지: 크레딧·캐시 데이터 임시 저장 (서버 전송 없음)

이용약관

본 서비스는 YouTube Data API v3를 이용하며 YouTube API 서비스 이용약관에 따릅니다.

채널 분석 데이터는 참고용이며, 투자·수익 결정의 근거로 사용하지 마시기 바랍니다.

무효 클릭 및 광고 부정 행위는 Google 정책 위반으로, 적발 시 서비스 이용이 제한됩니다.
