개인정보처리방침

본 서비스(YTAnalytics Pro)는 사용자의 개인정보를 수집하지 않습니다. YouTube Data API v3를 통해 공개된 채널·영상 정보만 조회합니다.

수집 및 이용 정보

  • 개인정보 수집: 없음 (비로그인 서비스)
  • 쿠키: Google AdSense 광고 목적의 서드파티 쿠키만 사용
  • 로컬스토리지: 크레딧·API 캐시 임시 저장 (서버 전송 없음, 브라우저에만 저장)

제3자 서비스

본 서비스는 Google AdSense 및 YouTube Data API를 사용합니다. 각 서비스의 개인정보처리방침이 적용됩니다.

  • Google 개인정보처리방침 →
  • YouTube API 서비스 이용약관 →

이용약관

본 서비스는 YouTube Data API v3를 이용하며 YouTube API 서비스 이용약관을 준수합니다.

면책 조항

채널 분석 데이터는 참고 목적으로만 제공됩니다. 투자·수익 결정의 근거로 사용하지 마십시오. 실제 YouTube 채널 수익은 조회수, 광고 단가, 국가, 시즌 등 다양한 요소에 따라 크게 달라질 수 있습니다.

광고 정책

본 서비스는 Google AdSense 광고를 사용합니다. 광고 무효 클릭은 Google 정책 위반이며, 적발 시 서비스 이용이 제한될 수 있습니다. 광고를 클릭하도록 타인에게 요청하는 행위를 금지합니다.

크레딧 시스템

하루 무료 크레딧(인기 영상 5회, 채널 분석 1회)은 자정에 초기화됩니다. 광고 시청 시 추가 크레딧이 지급되며, 이는 서비스 내 기능 사용권으로만 사용됩니다. 크레딧은 현금 또는 외부 혜택으로 전환되지 않습니다.
