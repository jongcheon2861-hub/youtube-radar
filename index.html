이용약관

본 서비스는 YouTube Data API v3를 활용하여 공개된 YouTube 채널 및 영상 데이터를 분석하는 도구입니다.

본 서비스는 YouTube의 공식 서비스가 아니며, Google LLC 또는 YouTube LLC와 관계가 없습니다.

  • 제공되는 분석 데이터는 참고용이며 실제 수익과 다를 수 있습니다.
  • 서비스 이용으로 발생하는 모든 결과에 대해 운영자는 책임지지 않습니다.
  • 크레딧 시스템은 API 할당량 보호 목적으로 운영됩니다.

개인정보처리방침

본 서비스는 사용자의 개인정보를 수집하지 않습니다. 검색 기록과 크레딧 정보는 사용자 브라우저의 localStorage에만 저장됩니다.

광고 서비스(Google AdSense)는 사용자 행동 데이터를 수집할 수 있으며, 이는 Google의 개인정보처리방침을 따릅니다.
